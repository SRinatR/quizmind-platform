# Database package

Prisma schema, migrations, and seed scripts live here.

## Current scope

The initial schema covers:

- identity and sessions;
- workspaces and memberships;
- plans, subscriptions, invoices, and payments;
- entitlement overrides and quota counters;
- feature flags and remote config versions/layers;
- extension installations, compatibility rules, and telemetry;
- audit, activity, security, domain, and support data.
